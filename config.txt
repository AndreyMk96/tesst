		Описание файлов

Файл server.py - содержит код серверной части 
Файл client.py - содержит код клиентской части 
Файл temp.py - содержит дополнительный код для работы серверной части
Файл список тестов.txt содержит описание проведенных тестов работы программы

		Руководство пользователя

Для запуска серверной части, необходимо запустить файл server.py
Для запуска клиентской части необходимо запустить файл client.py
Перед запуском файла client.py, необходимо выбрать тип запроса

В случае передачи post - запроса, в запросе в параметре "data" необходимо указать данные для записи
в параметре "params", необходимо указать параметр выбора бд для записи
При переданном параметре '1' - данные записываются в MySQL, при переданном параметре '2' данные записываютя в Redis.
Параметр надо передавать в кавычках

В случае передачи get - запроса, необходимо указать только параметр "params", для выбора данных для чтения
При переданном параметре '1' - данные считываются из MySQL, при переданном параметре '2' данные считываются из Redis,
При переданном параметре '3', данные считываются из JSON
Параметр надо передавать в кавычках
